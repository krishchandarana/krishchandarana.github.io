What is an RNN and LTSM?

An RNN stands for Recurrent Neural Network and it is a class of artificial neural networks where connections between units form a directed graph along a sequence. RNNs use their internal memory to process sequences of inputs.

An LTSM stands for Long Short-Term Memory, and it is a special type of RNN that uses special units in addition to standard units. LTSM units include a ‘memory cell’ which can contain the information in memory for long periods of time, and a set of gates is then used to dictate when information enters when it is outputted, and when it is forgotten.

How I Used Spatial Transformer Networks.

After realizing the problem, I trained a model in order to perform a spatial transformation on any image of a number between 1-9.

I used Pytorch in order to train the model to perform the spatial transformation on the images. The dataset is a public dataset, MNSIT. The dataset consits of a grid with images of numbers between 1-9, and the image may not be the easiest to interpret before the spatial transformation. The model is able to rotate, crop, and scale the image based on what is needed to make the number visually appealing to the user.
