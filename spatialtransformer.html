Spatial Transformer Network

A way for neural networks to enhance the geometric invariance of the model.

What Is A Spatial Transformer Network?

Neural Networks are not able to do anything to an image that you input.

That is a probelm as many of the times we need to crop a region of interst or scale the image. When we try to use neural networks to train a model for object detection, it may not be able to train if the picutre of a person is upside down. This can cause many problems to the final result of a model that is being trained.

In order to solve this problem you need to use a spatial transformer network (STN).What this does is it allows the neural network to do things like crop a region of interest in the image or even correct the orientation of the image.

The Solution

After realizing the problem, I incorperated a model in order to distinguish whether the skin cancer tumor was benign or malignant.

I used Tensorflow in order to train the model to distinguish between the two types of tumors. The dataset used was taken from the ISIC (International Skin Image Collaboration) Archive. The dataset consits of 1800 pictures of benign skin cancer tumors and 1497 pictures of malignant tumors. Throughout the code all the images have been scaled to a certain size, in order to make it easier for the model to get trained.

Implementing this model was such an awesome project, and I had a great time doing it. It was amazing to learn a new backend, Tensorflow as previously I have only used Pytorch.
